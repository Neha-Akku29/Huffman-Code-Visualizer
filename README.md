# Huffman Coding Visualizer

A frontend-based Huffman Coding project developed using HTML, CSS, and JavaScript that demonstrates the process of data compression using the Huffman Algorithm. The application allows users to enter text, generate Huffman codes, and visualize encoding results in an interactive and user-friendly interface.

---

## Features

- Text input for encoding
- Generates Huffman Codes dynamically
- Displays character frequencies
- Binary tree based compression logic
- Responsive and simple UI
- Real-time encoding visualization

---

##  Technologies Used

- HTML5
- CSS3
- JavaScript

---

##  About Huffman Coding

Huffman Coding is a lossless data compression algorithm used to reduce file size by assigning shorter binary codes to frequently occurring characters and longer codes to less frequent characters.

This project helps users understand:
- Binary Trees
- Greedy Algorithms
- Data Compression Techniques
- Character Frequency Encoding
